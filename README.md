# INTD-491_Perinatal-Depression-Screening-and-Management-Program
Class data science project focused on perinatal depression screening and management, using data analysis and digital tools to support early detection and informed decision-making

## Project Overview
This project explores how data science methods can support perinatal depression screening and management. The goal is to analyze relevant data, identify meaningful patterns, and design tools or insights that can help improve early detection and intervention.

## Objectives
- Explore and clean the dataset
- Analyze trends related to perinatal depression
- Build visualizations and/or predictive models
- Communicate findings clearly for decision-making

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## Repository Structure
- `data/` – datasets
- `notebooks/` – analysis notebooks
- `src/` – helper scripts
- `results/` – outputs, charts, reports
- `README.md` – project overview
